# Notes App

A full-stack notes application with CRUD functionality.

## Tech Stack

**Frontend:** React, TypeScript  
**Backend:** Node.js, TypeScript  

## Features
- Create, read, update, and delete notes. Add, delete tags for notes and filter notes based on tag.
